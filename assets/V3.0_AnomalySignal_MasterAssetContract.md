## II. 🧩 Component Breakdown & Logic Flow

### 1. Anomaly Signal Component (`<AnomalySignal />`)
**역할:** 논리적/정보적 위기(Knowledge Gap)를 시각화하는 핵심 트랜지션 및 오버레이 에셋.
**구현 방식:** React + SVG 기반 애니메이션 루프 (Lottie 또는 GSAP 연동).

*   **A-Signal Loop (Default):**
    *   **애니메이션:** 미세한 노이즈(Grain)가 배경을 스캔하며 무작위로 파열하는 효과. (Scanning/Flicker Pattern).
    *   **지속 시간:** 150ms ~ 400ms 간격으로 비동기적으로 발생.
    *   **트리거 지점:** 학술적 난제 제시 직후, 또는 핵심 개념이 깨지는 순간(Narrative Peak)에만 사용.
*   **V-CORE Collapse (State Change):**
    *   **애니메이션:** 텍스트나 데이터 구조가 갑자기 '파라미터 분해'되는 애니메이션. SVG의 `stroke` 속성이 0에서 무작위 값으로 급격히 변하며 시각적 충격을 준다.
    *   **트리거 지점:** 본편 영상의 클라이맥스, 또는 핵심 메커니즘이 실패하는 순간 (Mid-Roll/End-Credit).

### 2. Methylation Processor Module (`<MethylationProcessor />`)
**역할:** 생물학적 학술 개념을 시각화하는 인터랙티브 데이터 흐름 모듈.
**구현 방식:** SVG Path Animation + Gradient Morphing.

*   **기본 상태 (Idle):** 낮은 주파수의 파동(Waveform)이 주기적으로 움직이며 시스템 작동 중임을 암시한다. (Low-frequency, slow movement).
*   **활성화 상태 (Active):** 특정 단백질(Protein)에 결합할 때마다, 결합 부위가 **A-SIGNAL 색상**으로 강렬하게 하이라이트되며 주변 배경의 노이즈 패턴을 방해한다.
*   **데이터 Flow:** 텍스트 데이터나 개념적 연결 고리(Conceptual Link)는 점선 형태의 SVG Path로 표현되며, 이 경로를 따라 빛나는 에너지가 흐르며 정보를 전달하는 시각적 연출을 한다.

## III. 📐 Platform Specific Guidelines (Size & Behavior)

| 플랫폼 | 비율/크기 | 핵심 초점 | 필수 고려 사항 (개발자 지침) |
| :--- | :--- | :--- | :--- |
| **Desktop Player (YouTube)** | 16:9 (1920x1080 권장) | 깊이, 복잡성. 정보 밀도(Information Density)를 높인다. | 화면의 3분의 1 지점에는 반드시 'Authority Marker' (공식 인증 마크/로고)가 위치해야 한다. 시선 흐름을 중앙 하단으로 유도한다. |
| **Mobile Thumbnail** | 9:16 (1080x1920 권장) | 충격, 궁금증(Curiosity). 핵심 메시지를 단 하나의 비주얼로 압축. | A-SIGNAL 색상을 전면 배치하여 시선을 즉시 포획해야 한다. 복잡한 애니메이션은 썸네일 정지 이미지에서 명확하게 '암시'되어야 함. (예: 깨진 유리 조각의 패턴) |
| **Shorts Cover/Overlay** | 9:16 또는 1:1 | 속도, 즉각성(Immediacy). Hook에 최적화된 강렬한 대비와 움직임. | 오버레이는 배경과 분리되어 마치 '덧입혀진 정보 레이어'처럼 느껴지게 디자인해야 한다. 글자는 간결하고 크다 (Max 10단어 권장). |
