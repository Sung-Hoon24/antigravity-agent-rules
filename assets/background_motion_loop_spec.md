# Background Motion Loop Specification (Anomaly Cascade)
## [1] 목적: 콘텐츠 전체에 기술적 깊이와 긴장감을 부여하는 무한 배경 루프
## [2] 기본 모션:** 수많은 연결된 데이터 라인(Deep Indigo 기반). 일정한 흐름을 유지하며 전경을 가로지름.
## [3] 이벤트 로직 (Period 30s):
*   **Failure Event (T=0~1초):** 특정 구역의 데이터 라인이 `#FF4081` 색상으로 급격히 분산되거나, 그리드 패턴 자체가 일시적으로 왜곡(Warp)되는 시각적 충격을 발생.
*   **Recovery Event (T=2~3초):** 이 왜곡된 영역을 Stabilizing Cyan (`#4DD0E1`) 에너지 파장으로 덮으며 재정렬. 이전 루프보다 연결 강도가 높아진 것처럼 보이도록 마무리하여, 시스템이 '학습'한 느낌을 부여해야 함.
