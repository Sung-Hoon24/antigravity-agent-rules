# 🧚 팅커벨 5색 요정 가이드 시스템 원천 명세서 및 유실 복구서 (Tinkerbell Guide System Master Spec)

본 문서는 **[Rubia VFX: 튜닝 대시보드]**와 **[역분석 판독기 레이더]**에 탑재된 팅커벨 5색 요정 가이드 시스템의 모든 HTML 요소 ID, 요정별 멘트 DB, 그리고 자바스크립트 리스너 복구 코드를 안전하게 통합 보존하는 사양서입니다. 
향후 대시보드 업데이트나 파일 교체로 인해 가이드가 유실될 경우, 본 문서의 데이터와 코드를 활용하여 원상복구할 수 있습니다.

## 3. 🛠️ 자바스크립트 수립 및 팅커벨 가이더 복구용 코드 템플릿

대시보드와 판독기의 `<script>` 영역에서 아래의 코드 템플릿을 복사하여 즉각 팅커벨 바인딩을 격발시킬 수 있습니다.

### 3-1. 복구용 이벤트 바인딩 템플릿

```javascript
// 🧚 팅커벨 캐릭터별 호버 가이드 마운터 
const manualGuidesConfig = {
    'title-usage-guide': {
        girl: "소녀 조언 내용...",
        boy: "소년 조언 내용...",
        cat: "고양이 조언 내용...",
        squirrel: "다람쥐 조언 내용...",
        owl: "올빼미 조언 내용..."
    }
    // ... 추가 ID 맵
};

Object.entries(manualGuidesConfig).forEach(([id, quotes]) => {
    const el = document.getElementById(id);
    if (el) {
        el.addEventListener('mouseenter', () => {
            if (!isTinkerbellActive) return;
            const quote = quotes[currentCharacter] || quotes.girl;
            showTinkBalloon(quote);
        });
        el.addEventListener('mouseleave', hideTinkBalloon);
    }
});
```

### 3-2. 전역 툴팁 충돌 우회 가드 필터 (Bypass Filter)

공통으로 반복되는 자율 툴팁 오버레이 함수(`initAutoTinkerbellGuide`) 내부의 `interactiveSelector` 루프 초입에 아래와 같은 예외 가드 구문을 반드시 작성해 주어야 개별 지정 상세 멘트가 충돌로 인해 지워지지 않습니다.

```javascript
const interactiveSelector = 'button, select, input[type="range"], #recipe-drop-zone';
document.querySelectorAll(interactiveSelector).forEach(el => {
    if (el.hasAttribute('data-tink-tip')) return;
    
    // 📡 [한글 주석] 개별 상세 가이드가 존재하는 요소들은 공통 자동 순회 루프에서 제외 (충돌 방지 가드)
    if (
        el.id === 'btn-save-render-reverse' || 
        el.id === 'btn-register-template-reverse' || 
        el.id === 'btn-launch-analyzer' || 
        el.id === 'label-bg-image-upload' || 
        el.id === 'btn-copy' || 
        el.id === 'json-display' || 
        el.id === 'json-title-export' || 
        el.id === 'badge-intella-radar' || 
        el.id === 'text-vfx-integration' ||
        el.id === 'title-usage-guide' ||
        el.id === 'desc-reverse-why' ||
        el.id === 'title-steps-guide' ||
        el.id === 'list-steps-guide'
    ) return;

    el.addEventListener('mouseenter', () => {
        // ... (이하 공통 툴팁 로직)
    });
});
```

## 4. 🚑 유실 발생 시 5초 쾌속 복구 요령

1. 대시보드 및 판독기 HTML 파일 내부에서 마우스 호버 시 멘트가 뜨지 않는 요소의 ID가 본 규격서 1-1 또는 2-1에 등재된 ID로 지정되어 있는지 소스 분석을 통해 체크하십시오.
2. 만약 ID가 누락되었다면 태그에 `id="[지정ID]"` 형태로 삽입합니다.
3. 자바스크립트 블록 내의 `extraGuidesConfig` 또는 `manualGuidesConfig` 맵에 멘트가 채워져 있는지 본 명세서의 데이터를 비교 분석하여 누락된 캐릭터 멘트를 채우십시오.
4. 전역 툴팁 충돌 우회 조건식에 해당 ID가 잘 등재되어 있는지 교차 확인하면 즉각 복구가 완료됩니다.
