# Teaser Hook Script V3: The Void Trigger (7-15 seconds)

## 🎥 비주얼/사운드 사양서

*   **Tone:** Deep Indigo / Aged Gold. 불안정하고 권위적임.
*   **Pacing:** 매우 빠름 → 갑작스러운 느려짐(Void).
*   **BGM:** 미세한 노이즈와 함께 시작하는 앰비언트 사운드 (Ambient Noise Layering).

## 📝 스크립트 본문 (Voice Over)

**(0-3초: 빠른 속도로 정보를 폭격하며 시선을 고정)**
"우리가 '객관적'이라고 믿었던 모든 것은, 사실 수많은 편향의 필터를 거칩니다. 이 과정은 필수적이죠." (빠르게 진행)

**(3-7초: 톤 하락, 속도 저하. 배경 노이즈가 불안하게 떨림)**
"하지만 만약 그 '필터' 자체가 우리의 의지와 상관없이 *설계된* 하나의 모델이라면요? 우리가 아는 모든 것이... 완벽한 가설일 뿐이라면?"

**(7-10초: 클라이맥스 (The Void))**
(화면 전체가 Deep Indigo 노이즈로 뒤덮이며, 사운드가 찢기는 '시스템 에러'음을 폭발시킨 후 급격히 먹먹해짐.)
"우리가 경험하는 이 현실은... 과연 유일한 가능성입니까?"

**(10-15초: CTA/Hook)**
(질문이 화면에 텍스트로 강하게 남음. 배경 사운드가 다시 불안정하게 올라옴)
**[자막]: "우리가 놓치고 있는, 궁극의 '무엇(What)'은 무엇일까요?"**

**(CTA):** *전체 논쟁을 [날짜]에 공개되는 메인 영상에서 확인하세요.* (최종 2초 노출)
