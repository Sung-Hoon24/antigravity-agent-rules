# 📐 Funnel State Machine 컴포넌트 디자인 시스템 사양서 v2.0

## 🎯 목적 및 핵심 원칙 (Design Philosophy)
이 사양서는 콘텐츠의 조회수 증대가 아닌, 시청자의 **'정보 결핍(Information Gap)'**과 **'논리적 불확실성(Logical Uncertainty)'**을 극대화하여 유료 Funnel 진입을 강제하는 것을 목표로 합니다. 모든 컴포넌트와 애니메이션은 '학술적 권위'와 '시스템 오류의 불안정함'이라는 이중적인 톤앤매너를 유지해야 합니다.

*   **핵심 전략:** 논리적 전개 중 예상치 못한 시스템 경고(Error) 발생을 통해 시청자를 혼란에 빠뜨리고, 해결책을 외부(유료 Funnel)에서 찾도록 유도한다.
*   **톤앤매너:** Deep Indigo (신뢰/권위) + Aged Gold (지식의 가치) + Glitch Effect (불안정성/긴급함).

---

## 📊 Section 1: 레이아웃 그리드 시스템 (Grid System Specification)
YouTube 영상 콘텐츠(16:9 비율, 최소 해상도 1920x1080px 기준)에 최적화된 모듈형 그리드를 정의합니다. 모든 컴포넌트 배치는 이 그리드의 픽셀 단위 좌표를 준수해야 합니다.

*   **좌표 시스템:** Top-Left Corner (X=0, Y=0) 기준입니다.
*   **메인 콘텐츠 영역 (Main Content Zone):** `[X: 5% - 95%]`, `[Y: 10% - 90%]`
    *   가장 많은 정보와 시각적 집중이 이루어지는 핵심 영역. 모든 주요 애니메이션 컴포넌트는 이 영역을 벗어나지 않도록 배치되어야 합니다.
*   **시스템 경고/Funnel CTA 바 (Bottom Bar):** `[X: 0% - 100%]`, `[Y: 90% - 95%]`
    *   주요 전환(Transition) 지점에서 활성화되는 필수 컴포넌트가 고정 배치됩니다. 배경은 Deep Indigo의 미세한 노이즈 오버레이를 유지합니다.
*   **정보 결핍/하이라이트 구역 (Focus Spot):** `[X: 25% - 75%]`, `[Y: 30% - 60%]`
    *   논리적 모순이나 중요한 정보가 제시되는 순간, 시청자의 시선을 강제적으로 끌어당기는 영역. 이 구역에 '지식 간극'을 상징하는 오버레이(예: 왜곡된 글자 클로즈업)를 사용합니다.

## ⚙️ Section 2: 핵심 컴포넌트 사양 (Component Specifications)

### 1. [필수] ERROR_4001/시스템 경고 위젯
논리적 실패 지점(Logical Failure Point)에서 강제적으로 삽입되는 UI입니다. 단순한 화면 전환이 아니라, 시스템 자체의 결함처럼 보이게 연출해야 합니다.

| 사양 | 내용 (V2.0 최종 확정) | 동작 로직 |
| :--- | :--- | :--- |
| **배치** | Main Content Zone 전체를 덮는 Full-Screen Overlay. | 화면 전환 시점에서 즉시 활성화. |
| **컬러 코드** | Background: `#1A0B35` (Deep Indigo, 글리치 베이스). Text/Warning: `#FFD700` (Aged Gold, 불안정한 빛). | Deep Indigo와 Aged Gold의 대비를 통해 경고의 심각성을 강조. |
| **타이포** | Monospace Font (예: Courier New 또는 시스템 코드 폰트) 사용 의무화. 글자 크기 변동 및 왜곡 처리 필수. | 학술적 권위(코드)와 오류(왜곡)를 동시에 표현. |
| **애니메이션** | **[Start Frame]**: 화면 전체가 노이즈/글리치 패턴으로 휩쓸림 (0ms). $\rightarrow$ **[Transition]**: 글자 단위로 왜곡 및 깜빡임 발생 (50-150ms). $\rightarrow$ **[End Frame]**: "ERROR_4001: DATA INTEGRITY FAILURE" 메시지가 떨리며(Jitter) 중앙에 고정됨 (200ms 이후 유지). | *핵심:* 이 과정은 '실패'가 아닌, '더 깊은 곳을 탐색해야 함'이라는 암시를 주어야 합니다. |

### 2. [필수] 정보 결핍 오버레이 (Information Gap Overlay)
논리적 전개 중 시청자에게 의문을 강제하는 그래픽 장치입니다. 특정 정보가 제시된 직후, 그 정보를 '만족스럽게' 끝내지 않고 끊어내는 역할을 합니다.

| 사양 | 내용 (V2.0 최종 확정) | 동작 로직 |
| :--- | :--- | :--- |
| **배치** | Focus Spot 영역을 덮는 반투명(Opacity 70%) 레이어. | 특정 키워드/논리적 비약이 발생하는 순간 발동. |
| **비주얼 요소** | 빛나는 연결고리 그래픽 (Glowing Node Connectors) + 왜곡된 질문 마크(`?`) 클로즈업. | 지식의 '연결'을 상징하지만, 동시에 그 연결에 문제가 있음을 암시. |
| **애니메이션** | **[Start Frame]**: 배경 위에 미세한 스캔라인(Scanline)이 가로지르며 진입 (10ms). $\rightarrow$ **[Transition]**: '결핍'을 상징하는 물음표가 줌인되며 왜곡됨 (50-200ms). $\rightarrow$ **[End Frame]**: 화면 중앙에 "더 많은 정보가 필요합니다..."라는 문구가 Deep Indigo의 글리치 효과와 함께 나타남. | *핵심:* 시청자가 '아, 뭔가 빠진 게 있네?'라고 느끼게 만드는 것이 목표입니다. |

### 3. [필수] 전환 유도 CTA 위젯 (The Gatekeeper Widget)
정보 결핍 오버레이나 ERROR_4001 이후에 등장하여 다음 단계로의 이동을 막지 않고, '유료' 경로로 강제 이끄는 역할을 합니다.

| 사양 | 내용 (V2.0 최종 확정) | 동작 로직 |
| :--- | :--- | :--- |
| **배치** | 시스템 경고 바(Bottom Bar) 중앙에 팝업 형태로 위치. | 오버레이 또는 에러 발생 후, 시청자가 잠시 숨을 고르는 순간 (최소 3초 이상 노출). |
| **텍스트** | "이 정보의 완전한 해독은 다음 단계에서 가능합니다."와 같은 권위적 문구 사용. | 직접적인 '사기'가 아닌 '학술적 제한'처럼 보이게 해야 합니다. |
| **애니메이션** | **[Start Frame]**: 주변의 노이즈/글리치 효과를 흡수하며 등장 (Soft Fade-In, 100ms). $\rightarrow$ **[Transition]**: 마치 시스템 로딩 바가 채워지는 것처럼 CTA 영역 전체에 스캔라인 애니메이션이 흐름. $\rightarrow$ **[End Frame]**: '진단 워크북 다운로드' 버튼만 Aged Gold로 빛나며 가장 높은 주목도를 가짐 (Pulse Effect). | *핵심:* 이 위젯은 선택지가 아닌, 시스템의 필수 경고문처럼 작동해야 합니다. |
